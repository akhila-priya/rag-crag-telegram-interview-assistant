# Conversational RAG + CRAG Interview Assistant

An end-to-end AI-powered interview preparation assistant built using:

- Python
- LangChain
- OpenAI
- Pinecone
- Telegram Bot API

## Features
- Conversational memory
- Pinecone vector retrieval
- Corrective RAG (CRAG)
- Confidence score based broader retrieval
- Telegram UI

## Tech Stack
- Python
- LangChain
- OpenAI Embeddings
- Pinecone Vector DB
- Telegram Bot API
